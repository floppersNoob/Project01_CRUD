# DPWH Assets Folder

This folder contains DPWH branding assets for the Employee Profiling System.

## Recommended Files:

### Logo:
- `dpwh-logo.png` - Main DPWH logo (transparent background)
- `dpwh-logo-white.png` - White version for dark backgrounds

### Background Images:
- `dpwh-bg.jpg` - Background image for login page
- `dpwh-pattern.png` - Pattern overlay for backgrounds

## File Specifications:
- **Logo**: PNG format, transparent background, 200x200px recommended
- **Background**: JPG or PNG format, 1920x1080px recommended
- **Pattern**: PNG format, transparent background, seamless tile

## Usage:
These assets will be referenced in the login component:
```jsx
// Example usage in Login.jsx
<img src="/assets/images/dpwh-logo.png" alt="DPWH Logo" />
<div style={{ backgroundImage: 'url(/assets/images/dpwh-bg.jpg)' }}>
```
